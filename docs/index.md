# 🚀 backstage-app 문서

`backstage-app` 프로젝트의 기술 문서 사이트에 오신 것을 환영합니다. 이 문서는 Backstage의 TechDocs 기능을 사용하여 코드 저장소 내에서 직접 관리됩니다.

## 프로젝트 개요
이 카탈로그는 **backstage-test-1/backstage-app** 저장소를 기반으로 구축된 내부 개발자 포털(Internal Developer Portal)입니다. Kubernetes 환경에서 Backstage를 운영하며, GitOps 워크플로우를 통해 전체 시스템의 가시성과 운영 효율성을 높이는 것을 목표로 합니다.

## 주요 기능 및 연동 현황
* **Docs Like Code:** 마크다운을 통해 코드와 문서를 통합 관리합니다.
* **통합 카탈로그:** 서비스, 소프트웨어 템플릿, 문서를 한곳에서 관리합니다.
* **플러그인 연동 완료:** 
    * `backstage-app` 및 `backstage-manifests` 저장소에 대한 **GitHub** 및 **ArgoCD** 플러그인 연동이 완료되었습니다.
    * 이를 통해 배포 상태 확인 및 소스 코드 이력을 Backstage 인터페이스 내에서 직접 확인할 수 있습니다.

사이드바의 메뉴를 통해 아키텍처, 배포 프로세스 및 문제 해결 가이드를 확인해 보세요.
